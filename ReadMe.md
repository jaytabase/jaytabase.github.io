# ⚡ Lightning Fast Live Search with Vanilla JS ⚡
I have a PHP background and I've decided to move to JavaScript*\
This is my first attempt at creating something useful

### Vanilla JS live search
This "app" can dynamically filter a predefined list of data based on user input in real time, this works exceptionally quickly, and I've tested it with 1000+ records without seeing a noticeable drop in performance. Really useful to find your own information very quickly. I'm currently using this in an offline HTML file that I store some of my contacts in. Can easily be used for hundreds of different purposes.
